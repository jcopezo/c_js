Declaraciones y Sistaxis

Un programa de computadora es una lista de instrucciones a ser ejecutadas en un ordenador. En JavaScript esta lista de instrucciones se llama declaraciones, las cuales se ejecuta en el navegador web. Para que funcione correctamente hay que tener la sistaxis correcta, pues el programa puede mal interpretar. A continuacion vemos algunas items principales:


En JavaScript se utiliza punto y coma para separar sentencias y finalizar la instrucciones. Se puede poner diferente setencias en una sola linea:
var a, b, c;
a=5; b=6; c=a+b;

JavaScript ignora múltiples espacios. Por lo que las declaraciones a continuación son las mismas para el programa.
var t="Hola";
var t = "Hola";

JavaScript distinge entre mayúsculas y minúsculas, por lo que las variables VALOR, Valor y valor son diferentes.
var VALOR="Pezo";
var Valor=50.1;
var valor=true;

Para usar los caracteres especiales en JavaScript como tilde(á), diéresis(ü), entre otros, se debe usar la etiqueta <meta charset="utf-8">.

Lo escrito después de dos barras //Texto prueba o entre /*Texto prueba*/ es interpretado por JavaScript como un comentario y asi sea codigo no lo ejecutará. Generalmente son usados para crear comentario que indica la fucionaridad del código.
